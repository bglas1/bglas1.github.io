HTML

  1. Introdução
    1. O que é html?
      • HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar
        e apresentar o conteúdo de páginas web. É a base fundamental da web e define a estrutura
        e o layout dos elementos em uma página.
      • Ao escrever código HTML, você define a estrutura do seu documento, indicando onde os elementos devem ser exibidos na página.
        Posteriormente, você pode estilizar esses elementos usando CSS e adicionar interatividade por meio de JavaScript.
    2. Quem criou?
      • O criador foi um cientista da computação chamado Tim Berners-Lee, em 1990.
        Na época, Berners-Lee trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) e desenvolveu o HTML
      • Ele criou o HTML como uma linguagem simples de marcação, permitindo que os documentos fossem
        interligados por meio de links (hiperlinks). Com o tempo, o HTML evoluiu e se tornou a base da World Wide Web (WWW)
    3. Qual o objetivo dessa criação?
      • Tim Berners-Lee criou o HTML com o objetivo de facilitar o compartilhamento de informações e documentos científicos no CERN, onde trabalhava
      • Ele queria desenvolver uma forma eficiente de interligar documentos por meio de links,
        permitindo que os pesquisadores acessassem e compartilhassem informações de maneira mais rápida e fácil.
  2. Versões
    1. Primeira
      • A primeira versão do HTML foi o HTML 1.0, lançado por Tim Berners-Lee em 1993.
      • O HTML 1.0 foi bastante simples e limitado em comparação com as versões posteriores. Ele permitia a criação de cabeçalhos,
        parágrafos, listas, links e imagens, mas não incluía muitos recursos avançados de formatação e interatividade que temos hoje.
    2. Mais famosa
      • A versão mais conhecida e amplamente utilizada do HTML atualmente é o HTML5, lançada em 2014.
      • O HTML5 inclui novas tags (etiquetas) e atributos que tornam mais fácil a criação de elementos como áudio, vídeo,
        gráficos vetoriais, animações e formulários mais avançados. Além disso,
        ele oferece suporte a recursos de acessibilidade, como o uso de semântica para tornar o conteúdo mais claro para pessoas com deficiências visuais.
    3. Última
      • A última versão do HTML é o HTML5 também.
        Embora tenham sido lançadas algumas atualizações menores desde então, como o HTML5.1 e o HTML5.2,
        elas foram consideradas extensões e aprimoramentos do HTML5 original
      • O HTML5 é amplamente suportado pelos navegadores modernos e continua sendo
        a versão mais atual e recomendada para o desenvolvimento de páginas web.
  3. Linguagens de programação usadas no html
    1. CSS
      • Cascading Style Sheets
      • é usada para estilizar a aparência dos elementos na página, definindo cores,
        fontes, tamanhos, margens, posicionamento e outros atributos visuais
    2. JavaScript
      • JavaScript
      • é usada para adicionar interatividade às páginas, permitindo que os elementos na página
        respondam a eventos do usuário, como cliques, rolagem e digitação
    3. PHP
      • Hypertext Preprocessor
      • é usada para criar páginas dinâmicas que podem se comunicar com bancos de dados e gerar conteúdo personalizado para cada usuário.
    4. SQL
      • Structured Query Language
      • é usada em conjunto com o PHP para acessar e manipular bancos de dados.
  4. Onde usar o html?
    1. Sublime Text
      • O Sublime Text é um editor de texto avançado usado para escrever código em várias linguagens de programação,
        incluindo HTML, CSS, JavaScript e PHP. Ele é conhecido por ser rápido, leve e altamente personalizável,
        com recursos como realce de sintaxe, autocompletar, indentação automática e busca e substituição avançadas.
      • O Sublime Text também oferece suporte a plug-ins e pacotes adicionais que podem ser instalados
        para adicionar recursos adicionais ao editor. Ele está disponível para Windows,
        Mac e Linux e é amplamente utilizado por desenvolvedores em todo o mundo.
    2. Visual Studio Code
      • O Visual Studio Code é um editor de código fonte desenvolvido pela Microsoft.
        Ele é amplamente utilizado por programadores devido à sua interface intuitiva, recursos poderosos e extensibilidade.
      • Um dos principais atrativos do Visual Studio Code é a sua enorme biblioteca de extensões,
        que permite aos usuários personalizarem e estender as funcionalidades do editor de acordo com suas necessidades.
        Além disso, o Visual Studio Code é multiplataforma e está disponível para Windows, Mac e Linux.
    3. Atom
      • O Atom é um editor de código fonte gratuito e de código aberto desenvolvido pela equipe do GitHub.
        Ele é conhecido por sua flexibilidade, personalização e comunidade ativa.
      • Uma das principais características do Atom é a sua arquitetura baseada em pacotes,
        o que significa que os usuários podem estender e personalizar o editor através da instalação de pacotes adicionais.
        Isso permite que os desenvolvedores personalizem o ambiente de desenvolvimento de acordo com suas preferências e necessidades específicas.
  5. Criações mais famosas utilizando html
    1. Aplicativos
      • Uber: O aplicativo de compartilhamento de viagens mais popular do mundo foi criado usando
        HTML, CSS e JavaScript para a interface do usuário, enquanto o back-end é construído em outras linguagens de programação.
      • Netflix: O serviço de streaming de vídeo mais popular do mundo também usa HTML,
        CSS e JavaScript para a interface do usuário do site e aplicativo, permitindo que os usuários
        naveguem pelos títulos, assistam a trailers e gerenciem suas contas.
    2. Sites
      • Google (www.google.com): O motor de busca mais utilizado no mundo foi desenvolvido inicialmente usando HTML como parte da sua estrutura.
        Embora tenha evoluído ao longo dos anos, a base do site ainda é construída com HTML.
      • Facebook (www.facebook.com): A maior rede social do mundo também foi criada usando HTML como parte fundamental do seu desenvolvimento.
        O HTML é usado para estruturar e exibir o conteúdo, enquanto outras tecnologias e
        inguagens são utilizadas para adicionar recursos e funcionalidades adicionais.
    3. Páginas Web
      • Wikipedia (www.wikipedia.org): A conhecida enciclopédia online é construída principalmente usando HTML.
        A estrutura das páginas, o conteúdo e os links são todos formatados e exibidos através do HTML.
      • Amazon (www.amazon.com): Um dos maiores sites de comércio eletrônico do mundo, a Amazon, também utiliza HTML como parte fundamental da sua construção.
        O HTML é responsável por organizar e exibir os produtos, as informações do carrinho de compras e as diferentes seções do site.
